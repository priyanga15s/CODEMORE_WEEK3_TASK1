# CODEMORE_WEEK3_TASK1

Write a Java program that reads data from a file. Implement exception handling to catch 'FileNotFoundException' and 'IOException'. Display appropriate error messages if the file does not exist or if there is an error reading the file. This exercise will reinforce your understanding of handling checked exceptions.
